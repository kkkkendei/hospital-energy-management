# 医院能源管理系统 (Java 课设)

一个简单的医院能源管理系统，用于 Java 课程设计。

## 功能模块

1.  **用户管理**
    *   用户注册与登录
    *   角色管理（管理员、普通用户）
2.  **能源数据管理** (简化为手动录入和展示)
    *   能源数据录入
    *   能源数据查看
3.  **设备管理**
    *   设备信息录入
    *   设备信息查看

## 技术栈

*   **前端**: Vue.js (Vue CLI, Vue Router, Vuex, Element UI, Axios)
*   **后端**: Spring Boot (Spring Web, Spring Security, MyBatis/Spring Data JPA, MySQL)
*   **数据库**: MySQL
